**桌面清理带师**  

**这是什么？**  
这是一款为了解决国内电教委员们对于老师们全部文件往桌面丢的陋习的痛恨而编写的程序。  
这款程序能帮您定期删除或回收桌面的文件和文件夹，以帮助电教委员们远离c盘被炸的风险。  

**我要如何使用？**  
进入release页面，下载，双击，根据页面指引安装，没了:)  
软件默认安装目录为D:/Cleaner，暂不支持自行选择安装目录:(  
安装完后无需处理，静默运行。
